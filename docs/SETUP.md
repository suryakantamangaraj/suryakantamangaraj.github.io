# Project Setup Guide

## Prerequisites
- Node 18.18.0+
- Yarn
- Angular CLI 16+

## Quick Start
```bash
# Install Node 18
nvm install 18.18.0
nvm use 18.18.0

# Install Angular CLI
npm install -g @angular/cli@16

# Clone & Setup

cd portfolio-v2
